🚀 Meu Portfólio Profissional Interativo
Este projeto é um website estático e profissional desenvolvido para consolidar minha presença no mercado de tecnologia. Ele apresenta minha trajetória acadêmica na UNIVESP, minhas competências técnicas e os projetos que venho desenvolvendo.

📌 Sobre o Projeto
O objetivo principal foi construir uma vitrine digital utilizando apenas tecnologias fundamentais da web, sem o uso de frameworks de estilização, focando em semântica, acessibilidade e responsividade real.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo (header, main, section, footer).

CSS3: Estilização personalizada, uso de Flexbox para layout e Media Queries para responsividade.

Git & GitHub: Versionamento contínuo e documentação da evolução do projeto.

👤 Seções do Site
Home: Apresentação inicial e posicionamento como Desenvolvedor Web e Estudante de Engenharia.

Sobre Mim: Detalhes sobre minha formação no 6º semestre de Engenharia de Computação, histórico em informática e filosofia de aprendizado.

Projetos: Exposição de trabalhos práticos, como o Sistema Web CRUD com Django & MySQL.

Contato: Integração com LinkedIn, GitHub e link direto para e-mail.
