Trajetória e Competências

Sempre fui movido pela curiosidade tecnológica. Desde cedo, busquei entender o funcionamento das ferramentas digitais, o que me levou a realizar um curso de informática ainda na adolescência. Nessa formação, adquiri conhecimentos em softwares como Photoshop, Word e Excel, uma experiência fundamental que despertou minha proatividade e facilidade em dominar novas tecnologias.

Em 2020, tive um contato decisivo com o desenvolvimento web e lógica de programação através do curso de Informática para Internet. Foi ali que dei meus primeiros passos com HTML, CSS e C#, despertando a paixão por construir soluções reais através do código.

Hoje, canalizo toda essa experiência na graduação de Engenharia da Computação pela UNIVESP (6º semestre). O curso tem sido o pilar para aprofundar meus conhecimentos técnicos e me especializar em desenvolvimento web moderno.

Projetos em Destaque

🚀 Sistema Web CRUD | Django & MySQL(em construção)

Sistema de gerenciamento de tarefas robusto com autenticação de usuários, seguindo o padrão de projeto MVT (Model-View-Template). O projeto foca em segurança, organização de código e persistência de dados em banco relacional.

Destaques do Projeto:

  • Autenticação Segura: Sistema de cadastro e login de usuários.
  • Privacidade: Lógica de filtragem onde cada usuário gerencia apenas seus próprios dados.
  • CRUD Completo: Persistência total de dados com MySQL.
  • Interface Responsiva: Design adaptado para dispositivos móveis com Bootstrap 5.

Tecnologias: Python, Django, MySQL, Bootstrap e python-dotenv.
